# Church of Christ Anandapuram - Enterprise Refresh

This bundle keeps your original sections, content, gallery items, contact links and core interactions,
while upgrading the design to feel much more polished, structured and premium.

Files included:
- index.html
- gallery.html
- telugu.html
- gallery-te.html
- styles.css
- script.js
- favicon.svg
- site.webmanifest

What changed:
- Stronger enterprise-style visual system with better spacing, hierarchy and cards
- Premium hero section with clear calls to action
- Cleaner navigation, mobile menu and scroll progress bar
- Improved worship, ministry, program and contact presentation
- Richer footer and section intros
- Filterable gallery with metadata, keyboard-friendly lightbox and captions
- Image fallback placeholders so the site still looks complete even when image assets are missing
- Desktop-safe phone links that copy the number instead of triggering desktop calling apps
- Working alternate-language page shells so the language toggle no longer points to a missing file

Important note:
The uploaded bundle did not include the actual image assets referenced by the HTML. This refresh keeps the same filenames,
so if you place your real images in the same folder, they will show automatically.
